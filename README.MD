# Math and Logic Calculators

La primera versión de este proyecto fue desarrollada para la materia de **Matemáticas Discretas** de la carrera de **Ingeniería de Software de Jala University**.

El objetivo principal del proyecto es unificar en una sola aplicación web diversas herramientas y calculadoras que permitan automatizar procesos y facilitar el aprendizaje y la enseñanza de temas relacionados con matemáticas, lógica e ingeniería en general.

Actualmente, la aplicación se encuentra en desarrollo y cuenta con dos calculadoras funcionales. Además, incluye la estructura de diferentes herramientas que serán incorporadas próximamente.

El desarrollo del proyecto continuó posteriormente para la materia de **Cálculo 1** de la misma carrera y universidad, incorporando nuevas funcionalidades y ampliando el alcance de la aplicación.
